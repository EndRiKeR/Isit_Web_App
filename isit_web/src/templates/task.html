Базовая шаблонизация в Django для словаря. Создание базового интерфейса для просмотра списка (отели, товары, рейсы и тд) с ссылками и частью полей (цена, город) и при переходе по ссылке другая страница с более подробной информацией о товаре (описание, картинка и тд). В приложении должны быть использованы стили, для каждого элемента списка подгружается свое изображение. Добавить поле input для ограничения количества записей, отображаемых на странице (по умолчанию отображать все).
